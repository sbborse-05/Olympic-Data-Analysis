# 🏅 Olympic Data Analysis Web App

An interactive data analytics web application built using **Python, SQL, and Streamlit** to analyze historical Olympic datasets and extract meaningful insights.

---

## 📌 Project Overview

The Olympic Data Analysis Web App is designed to explore and visualize historical Olympic data including:

- Athlete performance trends
- Country-wise medal distribution
- Sport-wise dominance
- Year-wise participation growth
- Gender participation analysis

The application provides an interactive dashboard where users can filter data dynamically and gain insights in real time.

---

## 🚀 Features

✅ Interactive Streamlit Dashboard  
✅ Country-wise medal analysis  
✅ Athlete performance insights  
✅ Year and sport filtering  
✅ Data cleaning & preprocessing  
✅ Optimized SQL queries for faster data retrieval  

---

## 🛠 Tech Stack

- **Python**
- **Pandas & NumPy**
- **SQL (PostgreSQL / MySQL)**
- **Streamlit**
- **Matplotlib / Seaborn**
- **Git & GitHub**

---

## 📊 Key Insights Generated

- USA leads in overall medal count historically  
- Significant increase in female participation over the years  
- Certain sports dominated by specific countries  
- Medal trends fluctuate based on hosting country  

---

## 🧹 Data Preprocessing

- Handled missing values  
- Removed duplicate records  
- Standardized inconsistent country names  
- Optimized database schema for faster querying
